# Bem-vindo à Documentação da API Wegia

## Visão Geral

Este documento serve como a fonte central de informação para desenvolvedores que trabalham com a API do sistema Wegia. A API é construída sobre o framework **Laravel**, utilizando uma arquitetura robusta, modular e orientada a serviços para garantir escalabilidade e manutenibilidade.

O objetivo desta documentação é fornecer um guia claro e abrangente sobre como instalar, entender a arquitetura, utilizar a API e contribuir para o desenvolvimento do projeto.

## O que você encontrará aqui?

*   **[Guia de Instalação](instalacao.md):** Instruções passo a passo para configurar o ambiente de desenvolvimento localmente usando Docker.
*   **[Arquitetura do Software](arquitetura.md):** Uma explicação detalhada sobre a estrutura do projeto, incluindo a arquitetura em camadas (Controller, Service, Repository), o sistema de módulos e outros padrões utilizados.
*   **[Documentação da API](api.md):** Informações sobre como interagir com os endpoints da API, incluindo detalhes sobre autenticação, paginação, e como acessar a documentação Swagger.

## Pré-requisitos

Para começar a trabalhar com este projeto, é recomendado que você tenha familiaridade com as seguintes tecnologias:

*   PHP 8.x
*   Laravel Framework
*   Programação Orientada a Objetos (POO)
*   APIs RESTful
*   Docker e Docker Compose

Navegue pelas seções usando o menu para encontrar as informações que precisa.
