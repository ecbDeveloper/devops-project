# Assets

## Estrutura

- `assets/img/`: imagens internas empacotadas pelo Nuxt
- `public/`: arquivos estáticos públicos

## Arquivos identificados

`assets/img/`:

- `background-wegia.jpg`
- `logo_wegia.png`
- `sem_foto.png`

`public/`:

- `favicon.ico`
- `robots.txt`

## Uso no projeto

- Logo e imagens de fundo em telas como login e header
- Ícones majoritariamente via Font Awesome (CDN)
- Favicon dinâmico configurado em `app.vue` com dados de configuração
